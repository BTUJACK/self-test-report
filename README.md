# self-test-report
self-test report



针对word文档使用Python 3.8.7 版本执行自动化操作，并且用Python生成一个用户界面：
1
如果第二列某一行含有"："并且"："后面没有其他字符，隐藏第二列到第六列单元格的内框，只保留第二列到第六列单元格的外框，合并后的内容靠左对齐；禁止合并多行；

2
如果第二列第一个字符是数字第二个字符是小数点，隐藏第二列到第六列单元格的内框，只保留第二列到第六列单元格的外框，合并后的内容靠左对齐；禁止合并多行；
如果某一行执行了合并单元格的动作，把标题号码剪切到第一列：
标题号码的特点是：数字和点的组合（从第二列第一个数字开始直到第一个汉字结束）

3
如果某一行没有执行合并单元格的动作，把判定结果“符合要求”放在第三列；
4
如果某一行没有执行合并单元格的动作，在对应的第四列打勾；
5
